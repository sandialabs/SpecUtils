This is an example of how you can integrate the spectrum plotting into your HTML documents.
The interactions with the chart will essentually be the same as in InterSpec, just with no peak fitting and similar.

d3.v3.min.js: A feature-minimized build of d3 v3.5.17 — only the modules used by
  InterSpec's charts (SpectrumChartD3.js, D3TimeChart.js, etc.) are included.
  Stock d3 v3.5.17 minified is ~153 KB; this slim bundle is ~67 KB.
  To rebuild from upstream d3 source, see d3-slim-build/README.md.
